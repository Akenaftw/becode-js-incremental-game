3.3 Loading

Saving is cool! Very cool in fact! But kind of pointless if you can't load your save... To make it hassle free we will load the save in when the page is loaded. "let myCounter = localStorage.getItem("myCounter"); if (myCounter == null){ myCounter = 0} else{ myCounter = parseInt(myCounter)}" When the value that is pulled from the save is empty, we replace it with 0. When the value is available we want it to be parsed to an integer so we can calculate with it.
